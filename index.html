How It Works

Enter the Video Link
Drop the YouTube video URL into the box on the website.
Request a Transcription
Click the “Generate Transcript” button, and it will fetch the transcript from the YouTube video.
Read the Summary
Once the transcript is ready, you can view the summary on the Summary tab and save it as a PDF.

About This Project

Due to a busy work schedule, I found myself falling behind in my part-time web development classes at KodeGO. To address this challenge as part of my coursework (KodeGo-PP3), I developed a website that utilizes the YouTube API and Gemini API. The goal of this project is to create a tool that generates summaries of lesson recordings from YouTube, helping me catch up on missed content and enhance my learning experience while fulfilling the requirements of the KodeGo PP3 project.

If you have stumbled upon this site, I hope you find it useful! I have also documented the steps I took to create the site in the blog section. I hope you find the process insightful and helpful for your own projects. 🙂
